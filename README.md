# Étapes a suivre avant de lancer le projet

1. Créer un fichier `.env` à la racine du projet.
2. Y ajouter cette variable :
  - `NEXT_PUBLIC_API_URL=VOTRE_URL_API (ex: http://localhost:3000, sans le / a la fin)`

Une fois cela fait, rendez-vous sur `http://votre_url_api:3001/scraper/seed/races-and-classes` sur votre navigateur afin d'alimenter la base de données.
